Features

Everything you need,
nothing you don't

A focused tool built for one job: cleaning up your Android device safely and reversibly.

🔑
No Root Required

Works entirely over ADB using standard Android package manager commands. No SuperSU, no Magisk — just a USB cable.

↩️
Always Reversible

Uninstalls remove the app for your user only. The APK stays in the system image, so a single command (or factory reset) brings it back.

📋
Smart App List

Friendly names for 200+ common packages so you know what you're removing. Search, filter by type, and see system vs. user apps at a glance.

📱
Multi-device Support

Auto-detects connected devices, shows the model and Android version. Switch between multiple devices in the same session.

🔐
Anti-impulse Lock

Optionally require a password and a configurable wait period before restoration. Prevents accidentally restoring bloatware you deliberately removed.

🌍
Bilingual EN / AR

Full English and Arabic support with automatic right-to-left layout switching. Choose your language on first launch.

How it works

Simple, safe, transparent

1
Connect your device

Enable Developer Options on your phone, turn on USB Debugging, plug in the cable, and tap Allow on the authorization prompt.

2
Select apps to remove

Browse the app list, search by name or package, and tick what you want gone. Disable is lighter — the app is hidden but nothing is removed.

3
Restore any time

The Restore page lists everything you've removed. One click brings it back. If you set a time-lock, wait out the cool-down period first.

adb — under the hood
# Uninstall for user 0 only (keeps the APK)
$ adb shell pm uninstall -k --user 0
com.samsung.android.bixby.agent
Success
# Restore — APK is still in the system image
$ adb shell cmd package install-existing
com.samsung.android.bixby.agent
Package com.samsung.android.bixby.agent installed for user: 0
# Disable (lighter — no data loss)
$ adb shell pm disable-user --user 0
com.facebook.katana
